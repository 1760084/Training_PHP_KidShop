# Qui định đánh URL

**url: index.php?a=_1_&id=_2_**

* action:
    * a = 0: thông báo lỗi
    * a = 1: trang chủ
    * a = 2: danh sách sản phẩm theo hãng
    * a = 3: danh sách sản phẩm theo loại
    * a = 4: thông tin chi tiết của sản phẩm
    * a = 5: kết quả tìm kiếm sản phẩm
    * a = 6: quản lý giỏi hàng
    * a = 7: thanh toán
    * a = 8: đăng ký tài khoản
